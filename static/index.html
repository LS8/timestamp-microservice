Timestamp Microservice


What does it do?

  • A string can be passed as a parameter and it will check wether that string contains either an unix timestamp or a natural language date (e.g January 1, 1980)
  • If it does, both - the unix timestamp and the natural language date - will be returned
  • If it doesn't contain a recognizable date, null is returned for both properties

Example Usage:

https://timestamp-microservice-ls8.herokuapp.com/dec01
https://timestamp-microservice-ls8.herokuapp.com/1007164800

Example Output:

{"unix":1007164800,"natural":"Sat Dec 01 2001"}
